# Portfolio de Roberto García

Portfolio minimalista y limpio inspirado en tu CV.

## Incluye

- Presentación principal con nombre, título y resumen.
- Bloque lateral con contacto, educación, idiomas y habilidades.
- Experiencia y formaciones presentadas con estilo sobrio.
- Diseño responsive pensado para verse bien en escritorio y móvil.

## Personalización rápida

- Si quieres poner tu foto, añade un archivo llamado `assets/foto-perfil.jpg` y te ayudo a conectarlo en el avatar.

## Publicación

Puedes subirlo tal cual a GitHub Pages, Netlify o Vercel.
